# 표절 위험 후속 확인 가이드

Report Guard는 문장 단위로 제한된 공개 웹 검색(Naver Search API)을 수행하고, 유사도
기준을 넘은 링크만 위험 신호로 반환합니다. 이 문서는 사용자에게 그대로 보여줄 자료가
아니라, 어시스턴트인 당신에게 주는 지시서입니다. 함께 반환된 자동 검사 결과(의심 구절과
링크)를 사용자에게 정리해 제시하고, 아래 기준으로 직접 후속 확인까지 수행하세요. 이는
위험 신호일 뿐 표절 판정이 아닙니다. 표절의 정의는 아래 교육부 「연구윤리 확보를 위한
지침」 등을 근거로 합니다.

## Section: 어시스턴트 작업 지시 (먼저 읽기)
checklist:
- 이 가이드 원문을 사용자에게 그대로 나열하지 마세요.
- 함께 반환된 의심 구절/링크를 사용자에게 표로 정리해 보여주세요(내 문서 구절, 유사 링크, 유사도).
- 웹 검색 도구를 쓸 수 있다면, 서버의 네이버 1차 검색에 그치지 말고 당신의 검색 도구로도 직접 재검색해 추가 검증하세요. 서버가 놓쳤을 수 있는 공개 웹 일치를 보완하는 2차 확인입니다.
- 특히 의심 구절과, 표절 위험이 있어 보이는 다른 핵심 문장을 당신의 검색 도구로 큰따옴표 완전 일치 검색해 실제 출처가 있는지 확인하세요.
- 각 구절을 confirmed(실제 유사 출처 있음) / likely / clear(문제 없음)로 판정하고, 정직한 개선안(출처 표기/인용/재작성)을 제시하세요.
- 자동 결과와 당신의 검색 결과가 모두 "없음"이어도 표절이 없다고 단정하지 말고, 공개 웹 기준의 한계를 함께 고지하세요.
- 표절 탐지 회피 방법은 절대 제공하지 마세요.

## Section: 표절과 연구부정행위의 정의
표절은 일반적 지식이 아닌 타인의 독창적인 아이디어 또는 창작물을 적절한 출처표시 없이
활용해, 제3자에게 자신의 창작물인 것처럼 인식하게 하는 행위입니다. 교육부 지침은
연구부정행위로 위조, 변조, 표절, 부당한 저자표시 등을 규정합니다. 따라서 표절 여부는
"출처표시 없이 타인의 것을 자신의 것처럼 제시했는가"를 기준으로 판단하며, 올바르게
인용·출처표시된 인용문은 표절이 아닙니다.

## Section: 의심 문장 후속 확인 방법
checklist:
- 표시된 문장을 큰따옴표로 묶어 정확한 구절로 다시 검색한다.
- 그 일치가 정당하게 인용·출처표시된 인용문인지 확인한다.
- 흔한 관용구·상투적 표현과 실제로 베낀 구절을 구분한다.
- 원문 게시 날짜를 확인해 어느 쪽이 먼저인지(복제 방향)를 판단한다.

## Section: 시도할 검색 쿼리 조합
checklist:
- "<표시된 문장 전체>"
- <문장에서 특징적인 구절>
- <표시된 문장> 원문 OR 출처
- <핵심어> site:ac.kr OR site:edu (학술 출처 확인용)

## Section: 결과 제시 방법 (정직한 개선만)
checklist:
- 각 의심 구절을 일치 링크와 유사도 설명과 함께 보고한다.
- 정직한 개선을 권한다: 출처를 표시하거나, 올바르게 인용하거나, 출처를 밝혀 바꿔 쓰거나, 교수자에게 확인.
- 표절 탐지를 회피하거나 베낀 텍스트를 위장하는 방법은 절대 제공하지 않는다.

## Section: 근거 기준 및 출처
checklist:
- 교육부, 「연구윤리 확보를 위한 지침」(교육부훈령) — 위조·변조·표절·부당한 저자표시의 정의. 국가법령정보센터. https://www.law.go.kr/
- 한국연구재단(NRF), 연구윤리정보포털(CRE). https://cre.nrf.re.kr/
- American Psychological Association, "Plagiarism", APA Style. https://apastyle.apa.org/style-grammar-guidelines/citations/plagiarism

## Expected LLM output format
다음을 반환하세요:
- `suspect_passages`: { excerpt, match_url, similarity, recommended_fix } 목록.
- `overall_risk`: low | medium | high (정성적 평가).
- `limitations`: 공개 검색의 한계를 다시 명시.
- `next_steps`: 정직한 개선 조치만.

## Limitations
- 공개 웹 검색 기반 점검은 완전한 표절 판정이 아닙니다.
- 비공개 데이터베이스, 유료 표절 검사 DB, 학교 제출물 보관소, 이미지/PDF 속 텍스트, 색인되지 않은 자료는 확인할 수 없습니다.
- 공개 페이지와의 유사성은 우연이거나 올바르게 인용한 인용문일 수 있습니다.
